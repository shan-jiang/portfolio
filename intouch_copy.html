Team InTouch examined the highly social aspects of tabletop game interactions and mechanics. From there, we determined how these tabletop game elements can be adapted effectively into VR by developing a 3 - 5 minute multiplayer tabletop game that brings innovative social interactivity to the VR realm. Project official website: http://www.etc.cmu.edu/projects/intouch/
In this project I was responsible for pertinent literature review, prototyping and user tests. I also created and helped to implement interaction flow, user interfaces and 3D art assets by using Unity and Maya.
