Game TRAILER

This is a project aimed at making a multiplayer game with VR device. During the production of the game, we spent a lot of time on figuring out a mechanism for players to have fun in a strategic way and communicate with each other during the game. Havoc was selected to present at the Fall Festival of Entertainment Technology Center in Carnegie Mellon University as a game project.
My role in this project was designing the user interfaces for Web and creating 3D models. I also participated in game mechanism design process.

Problem & Solution



During the Fall Festival in Entertainment Technology Center, there would be more than 300 people coming to try our VR experience. Therefore, We wanted to figure out how to design and create an entertaining VR experience that allow multiple players to have fun at the same time, and also every turn could be finished in less than 5 minutes.



We decided to try combining VR and mobile devices together. We wanted to create an experience that allows one player to control VR and 2 to 6 players interact with her/him by using mobile devices. We used Internet to connect two sides and built a website for the game. The changes happened on website could be synchronize to the virtual world in HTC Vive.

Game Design

The Flow of Havoc

1. Mechanism

Basically, for mechanism, Havoc can be divided into two parts: VR part and Web part. Every round of the game lasts for 3 minutes. By the end of 3 minutes, if the core still has energy, then Mech protects it successfully. If the core has been drained out, then the nation which gets more points win the game.

VR part: The player needs to destroy towers to stop two nations from stealing the energy.

VR part Screenshot

Web part: Players need to make right angle triangles with two equal sides on the board in the web page to get energy.

Web part Screenshot

2. Playtest

We did playtests to see whether the game could be played as we expected. And during the process, we changed rules of the game to encourage players on the same team to communicate figuring out more interesting strategies while the VR player can try finding a wise way to defeat two nations. We also did tests for UI design and adjust the design for players to control more efficiently and intuitively.

User Interface Design

For Website interface Design, I executed user tests during playtests, changing the interfaces to a more efficient, intuitive and clear direction for users getting better experience when they engage in the game.
