Design Introduction



This is a set of GUI concept that I designed for a Smart TV. I used glowing effect and blur effect as key elements to create simple, aesthetically pleasing and unique interfaces for the new generation of entertianment television.

Design Details

Home page

Horizontal arrangement is convenient for TV remote control. Using different sizes of boxes for videos and Apps, easy for users to identify.



History page

History page is for users to go back to recent content quickly. When users switch between circles, contents will change for preview.
