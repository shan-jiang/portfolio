Global Game Jam × 2017 Award Winner

Game Video

Crowd Surfer was created in 48 hours during 2017 Global Game Jam in Pittsburgh. This Game won the JAMMER'S CHOICE award as the game receiving most votes from jammers among all games. I created 3D character models and textures for models. I also participated in game mechanism design part.
